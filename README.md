## 🚀 Features
- **Modern UI/UX**: Clean design with a focus on readability and typography.
- **Dynamic Pixel Hero**: Interactive pixel animation background.
- **Project Grid**: Data-driven project showcase powered by Markdown.
- **Dark & Light Mode**: Seamless theme switching with system preference support.
- **Fully Responsive**: Optimized for all devices and screen sizes.

## 🛠 Tech Stack
- **HTML5** & semantic markup.
- **Vanilla CSS** with modern features (CSS Variables, Flex/Grid).
- **Vanilla JavaScript** (ES6+) for interactivity and data fetching.
- **Asset Optimization**: SVG icons and optimized fonts.

## 📂 Project Structure
- `index.html`: Main entry point.
- `assets/`: Styling, images, and logic core.
- `data/`: Dynamic content (projects, etc.).
- `favicon.svg`: Vector-based site icon.
